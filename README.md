# Pulse Social Media Project

A multi-platform social media application (Web & Mobile).

## Repositories

### Backend
- pulse-backend: Node.js, MongoDB, Microservices

### Frontend Web
- pulse-frontend-web: ReactJS

### Mobile Application
- pulse-frontend-app: React Native

## My Role
- Backend Developer (worked with another backend developer)
- Sole Mobile Frontend Developer

## Key Features
- User authentication
- Post creation, likes, comments, follow users
- Real-time messaging, voice messages
- Video calling

## Technologies
Node.js, MongoDB, React, React Native, Socket.IO, JWT, Docker
